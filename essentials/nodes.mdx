## Node Configuration

Click or double click any node to open its configuration panel. Common fields include:

- **Variable name** ‚Äî How downstream nodes reference this node's output
- **Operation** ‚Äî Which action to perform (for multi operation nodes)
- **Input fields** ‚Äî The data the node needs to operate
- **Connection** ‚Äî Which credential to use (for nodes requiring OAuth)
