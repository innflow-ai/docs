## Notion

Read and write pages, databases, and blocks in Notion.

**Requires:** Notion OAuth connection

<Callout type="tip">
  To trigger a workflow when Notion content changes, use the [Notion Trigger](/triggers#notion-trigger) instead.
</Callout>

---

## Trello

Manage boards, lists, and cards in Trello.

**Requires:** Trello OAuth connection

---

## GitHub

Interact with GitHub repositories, issues, and pull requests.

**Requires:** GitHub OAuth connection

---

## Cal.com

Create and manage bookings with Cal.com scheduling.

**Requires:** Cal.com API credentials

---

## Calendly

Access scheduling data and events from Calendly.

**Requires:** Calendly API credentials
